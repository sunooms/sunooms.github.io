VS Code 的使用

vscode 设置成中文

vscode 默认的语言是英文，对于英文不好的小伙伴可能不太友好。简单几步教大家如何将 vscode 设置成中文。

  • 按快捷键“Ctrl+Shift+P”
  • 在“vscode”顶部会出现一个搜索框。
  • 输入“configure language”，然后回车。
  • vscode”里面就会打开一个语言配置文件。
  • 将“en-us”修改成“zh-cn”。
  • 按“Ctrl+S”保存设置。
  • 重启“vscode”，再次打开就可以看到中文界面了。

Prettier

比 Beautify 更好用的代码格式化插件

  • 1 打开 vscode 软件，方便快捷的方式就直接使用快捷键 ctrl+,（是 ctrl 加逗号）唤出快速搜索条界面。
  • 2 接下来先设定自动保存文件，搜索框贴入 files.autoSave 筛出设置项，并把设置项属性选择为 onFocuschange。
  • 3 设定编辑器默认代码格式化（美化）的插件为 Prettier，同理在搜索设置框贴入 editor.defaultFormatter，将配置项选择为 Prettier。
  • 4 设定 Prettier 插件保存时自动格式化代码，搜索设置项贴入 editor.formatOnSave，将如下图所示的选项框打钩即可。

可以配置.prettierc.js 文件,配合 eslint、git hooks 一起使用，提升项目代码的规范性

VS Code 同步配置的方法

一个编辑工具需要花费大量的时间去配置以及适应使用时间！ 例如把 vscode 配置成满意的方式，如果每换一次电脑就要重新来一次，大家一定会崩溃。放心，早就帮大家准备好了。可以爸 VSCode 的配置同步到网络上，这样可以随时优雅的迁移电脑环境，带走已经熟悉的 VSCode 配置，希望大家有所帮助！

资料来源“百度搜索”，这个都是公开的方法，参考多位作者的文章，就不列出原文作者。

同步配置整体流程：

1 在 vscode 上安装 settings sync 插件

在插件商店中搜索 Settings Sync，并安装。

2 登录

登录有两种方式，我一般选择 Github 登录，点击 LOGIN WITH GITHUB 。

这时候会自动打开默认的浏览器，我们点击 Authorize 用户名 进行授权。

3 上传配置信息

授权成功之后，使用 shift + ctrl + p 打开命名面板，输入 Sync，选择 "更新/上传配置" 。

这时候会自动创建一条 Gist 信息，并上传至 Github 。上传成功后，控制台会输出提示信息：

4 查看 Gists 信息

在浏览器中登录 Github，打开下拉菜单，选择 Your gists 。

可以看到刚才创建的 Gist 就已经出现了，点击就可以看到完整的配置信息。

5 下载配置

这时候问题就来了，如何在新的电脑中下载 Gist 配置信息呢？方法也非常简单，首先按上述方法进行登录，登录成功后会出现以下界面：

这里展示的是 Github 中所有的 ，一个 Gist 对应一个配置信息，选择你想使用的那个。这时候会把 Gist IF 自动配置到你的 VSCode 中。

然后再次打开命名面板，输入Sync，选择 下载配置 。

控制台同样输出相应的提示信息，这时候所有的插件和配置就已经同步过来啦！是不是 so easy！

6 总结

其实 VSCode 有自带的同步功能，在左下角的设置里面。但本人感觉不太好用，配置信息也不知道存在哪里。有兴趣的小伙伴可以自己试一试。

注意：如果在 Github 中打不开 ，那么你可能需要一些上网方法。
