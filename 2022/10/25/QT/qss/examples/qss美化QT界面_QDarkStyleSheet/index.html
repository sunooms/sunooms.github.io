QSS 美化 QT 界面

QSS 全称 Qt Style Sheets（Qt 样式表），用于美化 Qt 程序界面，类似于 CSS，但不如 CSS 强大(CSS2.0 的子集)，选择器和属性较少。

本文章主要介绍 QSS 的加载使用以及分享样式和编辑器，QSS 本身的语法详解请参考官方文档和其他教程。

QT 关于 QSS 的官方文档

Customizing Qt Widgets Using Style Sheets

Qt Style Sheets Reference

官方参考文档是最官方全面的文档，有时间最好仔细阅读一下。 主要包括：

  • 可应用样式表的控件列表
  • 属性列表
  • 图标列表
  • 属性值列表
  • 伪类选择器列表
  • 子控件控制列表

内容非常多，在此就不展开了。

基本语法

类似 CSS，QSS 每一条都是由一个选择器和一组声明构成： 选择器选出要对哪种控件进行样式修改， 每个声明都是键值对，键为属性，值为属性值

使用方式

为降低耦合，往往把 QSS 写在一个单独的 style.qss 文件中，然后在 main.py 的 QMainWindow 中加载样式

编写 QSS

新建一个扩展名为.qss 的文件，如 style.qss，编辑内容。（本文后面有完整的样式主题、QSS 编辑器推荐）

把写好的.qss 添加到 qrc 中

加载 QSS

有三种加载方式：

  • Widget 加载 QSS
  • QApplication 加载 QSS
  • 从文件中加载 QSS

具体见 加载QSS

QDarkStyleSheet

QDarkStyleSheet 是 github 上比较受欢迎的 QSS。

The most complete dark/light style sheet for Qt applications for Python 2*/3 and C++.

目前最新版本是 version 3, 支持 Python3( 3.6+)和 QT5。
